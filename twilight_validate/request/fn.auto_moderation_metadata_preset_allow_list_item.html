Ensure that an auto moderation rule’s allow_list item is correct when using presets.

The length must be at most AUTO_MODERATION_METADATA_PRESET_ALLOW_LIST_LENGTH_MAX. This is based on this documentation entry.

§Errors

Returns an error of type AutoModerationMetadataPresetAllowListItem if the length is invalid.
