Ensure that the stage instance’s topic length is correct.

The length must be at least STAGE_TOPIC_LENGTH_MIN and at most STAGE_TOPIC_LENGTH_MAX. This is based on this documentation entry.

Errors

Returns an error of type StageTopic if the length is invalid.
